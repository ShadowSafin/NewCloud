# NexxCloud Landing

Public marketing site for NexxCloud, built with Next.js App Router,
React, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validation

```bash
npm run lint
npm run build
npm audit --omit=dev
```

## Structure

- `src/app/` provides the route shell, metadata, and global design tokens.
- `src/components/` contains static sections and focused interactive controls.
- `public/hero-video.mp4` supplies the cinematic hero backdrop behind the product surface.

The page describes repository-backed capabilities: content-addressed storage,
resumable uploads, signed media delivery, integrity workers, Docker deployment,
Windows server packaging, the Windows desktop client, and the Android wrapper.
Release binaries and deployment instructions remain linked to the project repository.
